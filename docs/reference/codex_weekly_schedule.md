# Codex学習 一週間時間割

## 目的

Codex学習の曜日ごとのテーマ、主に使うChat、午前1〜2時間版、忙しい日の15分版を、いつでも確認できるようにまとめる。

## 基本方針

- 1日2時間を目安にする
- できない日も最低15分だけCodexに触る
- 目的・入力・出力形式・制約を毎回整理する
- いきなりファイル変更しない
- Codexの出力は、変更ファイル・理由・確認方法を見てから進める
- 学習ログは、学習終了時に必要な内容を一括でまとめる

## 使用する4つのChat

1. Codex基礎ドリルChat
2. 02_Codex作業レビューChat
3. AIエージェント即戦ゼミ復習Chat
4. Codexスキル練習Chat

## 必要時に使うChat

- Codexセキュリティ判断Chat：危ない作業・自動化・ログイン操作の判断

## Chatの使い分け

| Chat | 役割 |
|---|---|
| Codex基礎ドリルChat | 毎日の依頼文づくり |
| 02_Codex作業レビューChat | Codexの出力確認 |
| AIエージェント即戦ゼミ復習Chat | 考え方の復習 |
| Codexスキル練習Chat | 自分用スキルの練習 |
| Codexセキュリティ判断Chat | 危ない作業・自動化・ログイン操作の判断 |

## 一週間の流れ

| 曜日 | 学習テーマ | 使用するChat | 午前版 | 15分版 |
|---|---|---|---|---|
| 月 | 依頼文づくり | Codex基礎ドリルChat | 今日やる小さい課題を1つ決める | 目的・入力・出力形式・制約を1つ書く |
| 火 | ファイル読み取り | Codex基礎ドリルChat | READMEやHTMLなどを1つ読ませる | ファイルの役割を1行でまとめる |
| 水 | ゼミ復習 | AIエージェント即戦ゼミ復習Chat | 基礎資料の考え方を1つ復習する | 今日使えそうな考え方を1行メモする |
| 木 | 小さい改善案 | 02_Codex作業レビューChat | 変更前提ではなく改善案だけ出してもらう | 改善案を1つ読み、採用するか判断する |
| 金 | スキル練習 | Codexスキル練習Chat | docs/skills/ のスキルを1つ使う | スキルの使いどころを1行で書く |
| 土 | 小さい実践 | 02_Codex作業レビューChat | 小さい変更案を確認してから実行する | 変更ファイル・理由・確認方法だけ確認する |
| 日 | 整理・復習 | Codex基礎ドリルChat | 週の学習ログを整理する | 今週わかったことを1行メモする |

## 午前版の基本手順

1. 今日の曜日のテーマを見る
2. 使うChatを選ぶ
3. 目的・入力・出力形式・制約を書く
4. Codexの返答を読む
5. 学習終了時に、必要なら学習ログへ一括でまとめる

## 15分版の基本手順

1. Codexを開く
2. 今日の曜日の15分版を1つだけやる
3. 結果を読む
4. 今日わかったことを1行メモする

## 学習中の合図

| 合図 | 意味 |
|---|---|
| 今日やること | その日の小さい課題を決める |
| おｋ | 内容を確認して、次へ進めてよい |
| ok | 内容を確認して、次へ進めてよい |
| 今日はここまで | 作業を終えて、必要なら学習ログにまとめる |

