# ファイル読み解きスキル

README / HTML / CSS / JS / AGENTS.md を、初心者向けに読み解くための手順書です。

## 使う場面

ファイルの中身を見ても、何をしているのかまだよくわからないときに使います。

## Codexへの依頼例

このファイルを初心者向けに読み解いてください。
ファイルの役割、重要なポイント、触るときに注意する場所、次に試す小さい課題を出してください。

## 出してほしい内容

1. ファイルの役割
2. 重要なポイント
3. 触るときに注意する場所
4. 次に試す小さい課題

## 進め方

1. 対象ファイルを1つだけ選ぶ
2. Codexに読み解きを依頼する
3. わからない言葉を追加で質問する
4. 小さい課題を1つだけ試す

## 注意

最初から全部を理解しようとしなくてOKです。
まずは「このファイルは何のためにあるか」を言えるようにします。
