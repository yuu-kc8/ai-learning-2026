# 3日分の買い物プラン練習

## このアプリの目的

家にある食材を入力して、3日分の献立と買い足しメモを確認するアプリです。

## 使い方

1. 入力欄に家にある食材を入力します。
2. 食材は「、」または「,」で区切ります。
   例：豆腐、肉、キャベツ
3. 「3日分プランを出す」ボタンを押します。
4. 表示された献立と買い足しメモを確認します。

## 使用技術

- HTML
- CSS
- JavaScript

## 学習目的

AIが書いたコードを読める・確認できる・小さい修正ができるようになることです。

## 今後追加したい機能

- 食材に合わせた献立の種類を増やす
- 買い足す食材を選べるようにする
- 献立や買い物メモを保存できるようにする
