## 3日分の買い物プラン練習

家にある食材を入力すると、3日分の献立と買い足しメモを表示する練習用ミニアプリです。

できること：
- 食材を「、」や「,」で区切って入力できる
- 入力した食材を一覧で表示できる
- 4個目以降の食材を「予備の食材」として表示できる
- 家にある食材と同じものは買い足しメモから外せる
- 3日分の献立に、入力した食材を反映できる
